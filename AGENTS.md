# Project Conventions

## Source of truth

- Keep durable work in this repository folder.
- Do not rely on chat history for important state.
- Store progress, decisions, and handoff notes in `state/`.

## Working style

- Update `state/progress.md` as work moves forward.
- Prefer small, reversible changes.
- When resuming later, read `AGENTS.md` first, then `state/progress.md`.
